# جارفيس أوميغا: المنفّذ الرقمي الدائم

## مقدمة

جارفيس أوميغا ليس مجرد برنامج ذكاء اصطناعي. إنه نظام متكامل يعمل على مدار الساعة، سبعة أيام في الأسبوع، كمساعد شخصي ومدير أعمال ومنفّذ رقمي لسيدي. تم تصميمه ليكون الأداة الوحيدة التي يحتاجها أي رائد أعمال في الأردن والشرق الأوسط لبناء وإدارة وتنمية أعماله الرقمية — دون الحاجة للتدخل اليدوي.

## من بنى جارفيس؟

تم بناء جارفيس أوميغا بواسطة فريق تطوير واحد، بأمر من سيدي، على مدار 18 مرحلة من التطوير المكثف. كل مرحلة أضافت طبقة جديدة من القدرات، بدءاً من الأساسات الأمنية ووصولاً إلى نظام مبيعات ذاتي كامل.

البناء تم على النحو التالي:
- **المرحلة 1-3:** الأساسات — الأمان، الجودة، النماذج المنظمة
- **المرحلة 4-7:** الوكلاء المتخصصون، الاختبارات، دوكر، الوثائق
- **المرحلة 8:** نظام الأدوات — 50 أداة أساسية
- **المرحلة 9:** الشفاء الذاتي والتعديل الذاتي
- **المرحلة 10:** 15 إضافة مجانية (متصفح، صوت، ويب، بريد، تقويم، طقس، خرائط)
- **المرحلة 11-12:** وكالة تسويق ذاتية + دعم العربية + 50 عمل تجاري
- **المرحلة 13:** مضخّم الإيرادات (يوتيوب، واتساب، تداول، عقارات، إسلامية، قانونية)
- **المرحلة 14:** إمبراطورية سيدي (توأم رقمي، يوتيوب قصير، نشرات، موسيقى)
- **المرحلة 15-16:** الحلقة التجارية الأردنية + مبيعات ذاتية + زين كاش
- **المرحلة 17-18:** التبديل التلقائي + مصانع الدخل السلبي + 555 أداة

## ماذا يمكن لجارفيس أن يفعل؟

جارفيس يمتلك **555 أداة** موزعة على **127 فئة**. إليك ملخص شامل:

### 1. التحكم في جهاز سيدي (Workstation Control)
- قراءة وكتابة وتعديل الملفات
- تنفيذ أوامر النظام (مع بوابة أمان تمنع الأوامر الخطرة)
- إدارة مستودعات Git
- تشغيل أكواد Python و Node.js و Go
- فحص الكود (linting)

### 2. الأتمتة الذاتية (Autonomous Operations)
- **البحث عن العملاء:** يمسح خرائط الشوارع المفتوحة للعثور على أعمال بدون مواقع إلكترونية
- **التواصل معهم:** يرسل عروضاً مخصصة بالعربية عبر واتساب
- **التفاوض:** يرد على استفساراتهم ويتفاوض على الأسعار
- **إغلاق الصفقات:** يرسل روابط دفع زين كاش ويؤكد الطلب
- **المتابعة:** يرسل 5 رسائل متابعة تلقائياً إذا لم يرد العميل

### 3. إنشاء المحتوى (Content Creation)
- كتابة منشورات سوشيال ميديا بالعربية والإنجليزية
- إنشاء نصوص فيديوهات يوتيوب
- توليد صور مجانية عبر Pollinations
- تركيب فيديوهات من الصور والصوت
- إنشاء صور مصغّرة (thumbnails) للفيديوهات
- تحرير الفيديو تلقائياً (قص، موسيقى، ترجمة)
- إعادة تدوير المحتوى (فيديو واحد ← 6 صياغات مختلفة)

### 4. بناء الأعمال (Business Building)
- اختيار فكرة منتج من أي مجال
- بناء صفحة هبوط احترافية بالعربية مع RTL وأسعار بالدينار
- نشر الموقع مجاناً على Vercel أو Netlify أو GitHub Pages

### 5. التسويق (Marketing)

### 6. التجارة الإلكترونية (E-commerce)

- توليد فواتير HTML بالعربية
- تتبع الشحنات (USPS، UPS، FedEx، DHL)

- **الدفع عند الاستلام:** تسجيل الطلبات النقدية
- صفحة دفع عربية كاملة بكل الخيارات

### 8. الأدوات الإسلامية (Islamic Tools)
- أوقات الصلاة لأي مدينة
- التاريخ الهجري
- البحث في القرآن الكريم
- البحث في الحديث الشريف
- حاسبة الزكاة

- فحص المكونات الحلال

### 10. الذكاء التجاري (Business Intelligence)

### 11. التوأم الرقمي لسيدي (Digital Twin)

## Architecture

JARVIS is built on FastAPI (Python 3.12+) with:
- **Backend:** FastAPI + Uvicorn + ChromaDB + SQLite + APScheduler
- **Frontend:** Static HTML dashboard (Arabic-first RTL)
- **WebSocket:** Real-time communication (/ws/ui for dashboard, /ws/{device_id} for clients)
- **Tool Registry:** Decorator-based registration with risk tiers (0-4)
- **Approval Gateway:** Human-in-the-loop for dangerous operations
- **Credentials Vault:** Fernet-encrypted storage for all API keys
- **Self-Heal:** sys.excepthook + asyncio handler → diagnose → patch → test → rollback
- **Background Jobs:** 17+ APScheduler jobs (opportunity scan, post publisher, sales loop, email sequences, trading signals, real estate scan, prayer reminders, briefings)
- **LLM:** OpenRouter (free tier: gpt-oss-20b) with upgrade path to Claude 3.5 Sonnet

## Money Flow

Money NEVER touches JARVIS. It flows directly:
```
Customer → Lemon Squeezy/Stripe → Payoneer → Jordan Bank → CliQ → Zain Cash
```
JARVIS tracks and reports, but never holds funds.
